About HTML

HTML stands for Hyper Text Markup Language. HTML is the standard markup language for Web pages. HTML elements are the building blocks of HTML pages.

HTML elements are represented by '<>' tags. Web developers use HTML keywords or tags to instruct the Web browser application how to format and display the content of Web pages.

HTML is a very evolving markup language and has evolved with various versions updating.
