About C#

C# is a modern language.It was developed around 2000 by Microsoft as part of its .NET initiative and later approved as an international standard by Ecma (ECMA-334) in 2002 and ISO (ISO/IEC 23270) in 2003. Mono is the name of the free and open-source project to develop a compiler and runtime for the language.The first version was released in 2001.The most recent version is C# 8.0, which was released in September 2019.

What is C# ?

C# is pronounced as"C-Sharp" .C# tutorial provides basic and advanced concepts of C#.C# is a programming language of .Net Framework.

C# is approved as a standard by ECMA and ISO. C# is designed for CLI (Common Language Infrastructure). CLI is a specification that describes executable code and runtime environment.C# programming language is influenced by C++, Java, Eiffel, Modula-3, Pascal etc. languages.In Csharp Langauge file extention is ".cs".
